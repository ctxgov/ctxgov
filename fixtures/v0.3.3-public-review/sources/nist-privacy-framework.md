# Privacy risk management

The NIST Privacy Framework is a voluntary tool for identifying and managing
privacy risk. For the CtxVault public review pack, this source represents the
need to inspect privacy impact before context is reused or projected.

# Handoff relevance

A safe context handoff should show which selected sources may affect privacy,
what preflight decision was made, and where the receipt evidence lives. This
source should be selected for queries about privacy preflight, enterprise risk,
receipt inspection, selected context, and governed release review.
