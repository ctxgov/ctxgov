# CtxGov v0.6.12 - Live Link Verifier

CtxGov v0.6.12 adds an optional network-enabled live-link verifier for public
release integrity checks after offline release-pack checks pass.

## Shipped

- `scripts/check_public_live_links.py`
- Offline unit tests for the verifier contract.
- Public release notes for the network-enabled verification boundary.

## Boundary

No public benchmark claim. No security guarantee. No provider/model call. No
adoption claim. No package publication. No hosted runtime or live adapter. No
CLI beta.

The verifier is opt-in and checks public URLs only.
