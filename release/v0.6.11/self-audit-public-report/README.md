# Self-Audit Public Report

This pack records the v0.6.11 self-audit of CtxGov public-surface drift after
the v0.6.9 and v0.6.10 public releases.

## Included Files

- `self-audit-summary.json` - machine-readable finding summary.
- `claim-lint.json` - public claim-boundary receipt.
- `leak-scan.json` - public leak-scan receipt.
- `link-check.json` - local and live-link receipt.
- `publication-readiness.json` - owner approval and public-write scope.
- `publication-execution-receipt.json` - publication receipt.
- `manifest.json` - release-pack inventory.

## Boundary

This self-audit is not a benchmark result, security finding, adoption claim,
provider/model compatibility claim, package publication claim, hosted runtime
claim, live adapter claim, stable ASCR standard claim, or Memory X-Ray CLI beta.
