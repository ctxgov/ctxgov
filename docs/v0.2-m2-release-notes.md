# CtxVault v0.2/M2 Release Notes

Status: v0.2.0 public developer-framework milestone.

## Positioning

v0.2 is a developer-framework milestone for the inspectable local context core.
It is not a stable v1 release.

## Added

- read-only projection adapter healthchecks for `AGENTS.md`, `CLAUDE.md`, and
  workstream briefs
- runtime event receipts in projection healthcheck output
- optional local backup write flow with snapshot creation, sync manifest copy,
  replica verification, and a local backup write receipt
- CLI command `local-backup-write`
- MCP tool `local-backup.write`
- public trademark and official-project usage policy

## Preserved

- deterministic local core
- review-gated promotion semantics
- projection receipts for generated outputs
- Apache-2.0 public core license
- clear private/public boundary for first-party workbench, wrapper source,
  signing operations, release operations, and brand assets

## Experimental

- projection healthcheck contract
- runtime event receipt contract
- `Episode`, `Workstream`, and `intelligence.py`
- plugin manifest and projection receipt contracts

## Not Included

- hosted sync or cloud backup
- public Forgejo, restic, or git-bundle product contracts
- live connector guarantees
- automatic model-driven promotion
- stable v1 compatibility guarantees
