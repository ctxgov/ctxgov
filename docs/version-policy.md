# CtxGov Public Version Policy

The v0.8.0 public package uses one current public version: `0.8.0`.

| Dimension | Current public value | Scope |
| --- | --- | --- |
| Product version | `0.8.0` | Public README, release notes, Pages copy |
| Package version | `0.8.0` | Public extracted source package metadata |
| Evidence artifact | `release/v0.8.0/` | Public package evidence and receipts |
| Schema identifiers | `ctxvault.*` legacy strings | Stable contract IDs, not branding |

Historical release identifiers are archive material and must not be presented as
the current v0.8.0 product or package version.
