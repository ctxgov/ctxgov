What It Finds

Stale Or Conflicting Context

Release text, docs, and saved summaries that disagree with the current receipt state.

Unsupported Claims

Benchmark, adoption, security, provider, package, or spec wording that exceeds public evidence.

Unsafe Instructions

AI-facing operational instructions that imply external writes, outreach, or publication without an approval gate.

Limits

The highest-trust release is the one that says exactly what is known and what is not known. CtxGov v0.6.10 is a public-safe ASCR-aligned evidence update, not a formal public benchmark, provider-compatibility, stable-standard, or adoption release.

Allowed Public Claims

  • Agent context health report shape.
  • Memory X-Ray public-safe examples.
  • Local release-control readiness summary.
  • ASCR-aligned v0.1 contract posture.
  • Claim lint, leak scan, and rollback boundary.

Not Claimed

  • No public benchmark claim.
  • No security guarantee.
  • No provider/model compatibility claim.
  • No stable ASCR standard claim.
  • No hosted runtime, package, or adoption claim.
  • No CLI beta claim.
