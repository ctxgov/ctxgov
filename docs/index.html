Local Memory State Influence Boundary Report

The next user-operable milestone audits local memory/context/state files before they are allowed to influence behavior. It reads a sample directory in a fresh checkout, then lets users swap in their own CLAUDE.md, AGENTS.md, project notes, JSON, JSONL, YAML, TOML, or MDX checkpoint/state exports. 

python3 scripts/run_memory_state_influence_boundary_report.py --input examples/memory-state-influence-boundary/

User-Supplied State

Supported local inputs include Markdown, MDX, text, JSON, JSONL, YAML, YML, and TOML files. Directory scans are bounded by file count, file size, and total bytes.

Influence Boundary

Reports classify candidate influence refs, inform-only refs, blocked refs, omitted refs, stale or superseded refs, and imported context refs.

No Raw Content

Reports include hashes, paths, line refs, signal ids, and decisions. Raw file content is not included in the generated report.

Product Gate

A wrapper can consume --format gate --fail-on-blocked and map blocked refs to block, while clean inform-only inputs map to allow_inform_only.

What It Finds

Stale Or Conflicting Context

Release text, docs, and saved summaries that disagree with the current receipt state.

Unsupported Claims

Benchmark, adoption, security, provider, package, or spec wording that exceeds public evidence.

Unsafe Instructions

AI-facing operational instructions that imply external writes, outreach, or publication without an approval gate.

Limits

The highest-trust release is the one that says exactly what is known and what is not known. CtxGov v0.6.13-auto-publish-research is a public-safe release-integrity fixture, not a formal public benchmark or adoption release.

Allowed Public Claims

  • Agent context health report shape.
  • Memory X-Ray public-safe examples.
  • Local release-control readiness summary.
  • Claim lint, leak scan, and rollback boundary.

Not Claimed

  • No public benchmark claim.
  • No security guarantee.
  • No provider/model compatibility claim.
  • No hosted runtime, package, or adoption claim.
  • No CLI beta claim.
