CtxGov Task Shard Context Budget

Audit a saved long-agent task plan locally. The report shows which context each shard may see, which refs were omitted, which refs are searchable-only, which events were compacted, and where budget, cache, merge, or replan risk needs review.

Run Locally

python3 scripts/run_task_shard_context_budget_demo.py --input examples/task-shard-context-budget/

What The Report Covers

Field Purpose
selected context refs Refs admitted into the shard's local context budget.
omitted context refs Refs kept visible as omitted evidence rather than silently lost.
searchable-only refs Refs that may be rediscovered but are not injected as authority.
compacted event refs Summary boundaries that may hide detail and need source recovery.
per-shard token budget Local budget pressure per task shard.
cache-hit-is-not-authority rule Cache reuse can lower cost, but grants no answer or action authority.
merge/replan gates Downstream reliance requires explicit review evidence.
blocked side effects No workflow, worktree, provider, target, or memory backend side effects.
