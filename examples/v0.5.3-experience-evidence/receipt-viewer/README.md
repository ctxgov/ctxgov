# Receipt Viewer Sanitized Example

Status: v0.5.3 public sanitized example.

Governed context projection for AI work.

Boundary phrase for publication review: no target repository writes and no provider/model execution.

This example shows a redacted receipt inspection result. It is synthetic and
does not include private paths, raw private receipts, raw source excerpts,
provider/model outputs, or target repository writes.

## Files

- `receipt-inspection.json`: sanitized receipt inspection payload.
- `manifest.json`: source template, omitted refs, boundary, publication status,
  and rollback.

## What It Shows

1. The viewer is read-only.
2. Raw receipt payload is hidden by default.
3. Selected, omitted, blocked, warning, and rollback fields are visible.
4. Side-effect booleans remain false.

## Not Claimed

Do not use this example to claim benchmark results, reliability improvement,
adapter support, runtime support, provider/model compatibility, hardware/cost
benefits, automatic repository optimization, stable Memory Governance
Protocol, Memory OS behavior, RAG replacement, hallucination prevention, or
security certification.
