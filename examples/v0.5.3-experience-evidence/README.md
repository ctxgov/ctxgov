# v0.5.3 Experience Evidence Pack

Status: v0.5.3 public sanitized example.

Governed context projection for AI work.

Boundary phrase for publication review: no target repository writes and no provider/model execution.

This pack promotes the v0.5.2 experience-surface examples into a public
v0.5.3 evidence pack. The examples are synthetic and sanitized. They show how
reviewers can inspect redacted receipt state and a Future Context Review queue
before context becomes agent-visible.

## Examples

- `receipt-viewer/`: redacted receipt inspection surface.
- `future-context-review/`: risk-ranked Future Context Review queue.

Each example keeps selected, omitted, blocked, warning, side-effect, and
rollback fields visible. Raw private receipts, private paths, raw source
excerpts, provider/model outputs, target writes, and benchmark claims are not
included.

## Review Boundary

Use this example pack to explain context inspection, review state, and receipt
boundaries. Do not use it to claim benchmark results, reliability improvement,
adapter support, runtime support, provider/model compatibility, hardware/cost
benefits, automatic repository optimization, stable Memory Governance
Protocol, Memory OS behavior, RAG replacement, hallucination prevention, or
security certification.
