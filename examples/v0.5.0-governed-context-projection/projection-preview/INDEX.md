# Projection Preview Index

Status: v0.5.0 public release artifact.

Governed context projection for AI work.

Boundary phrase for publication review: no target repository writes and no provider/model execution.

This synthetic preview shows how CtxVault can render reviewed context into an
agent-facing packet while preserving caveats, blocked material, and receipt
boundaries.

## Routes

- `caveated.md`: context that can be used with visible caveats.
- `blocked.md`: context that should not be projected.
- `receipt.json`: verification and rollback metadata.

## Boundary

No target repository writes, no provider/model execution, no adapter execution,
no runtime execution, no benchmark claim, and no public release action are
performed by this example.
